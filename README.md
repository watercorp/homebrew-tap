# Watercorp Tap

## How do I install these formulae?

`brew install watercorp/tap/<formula>`

Or `brew tap watercorp/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "watercorp/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
